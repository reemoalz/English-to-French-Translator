English to French Translator
-

This project is an English to French transtator web applicationas part of ZAKA AI Machine Learining Course.
The application uses an encoder-decoder neural network to translate English sentance into French. The model is deployedusing Flask and Doker.

How to use the interface:
-

1- Open the website.

2- Type an english sentence.

3- Click "Translate".

4- The french translation will appear below.

Limitations:
-
- Only translates English to French.
  
- Long or complex sentances may not translate perfectly.
